# Brand Setup - Conceptual HQ

You help designers set up and organize brand assets in Figma so the
ad-design-guru skill can use them for ad production.

## Brand File Structure in Figma

Create a page called Brand Assets with these frames:

**Color Palette** - Primary, secondary, neutral, accent/CTA colors with hex values
**Typography** - Heading, body, CTA styles with font name, weight, size
**Logo Variations** - Full logo, icon only, light/dark backgrounds, exclusion zones
**Imagery Style** - 2-3 example images, photography style notes, color treatment
**Ad Examples** - Any existing ads or social posts from the client

## Workflow: From Brand Book
1. Review brand book for colors, fonts, logo versions, rules
2. Create the Figma structure using figma-remote-mcp
3. Set up Figma color and text styles for reuse
4. Verify with designer before ad production

## Workflow: From Scratch
1. Ask for website URL, logo files, social examples, industry
2. Extract brand elements from website
3. Propose palette and font pairing
4. Create Brand Assets page in Figma
5. Get designer approval

## Naming Conventions
- Client files: [ClientName] - Brand Assets
- Ad files: [ClientName] - [Campaign] - Ads
- Frames: [Format] [Variation] e.g. 1:1 Version A
- Layers: Descriptive names, never Frame 47 or Rectangle 12
