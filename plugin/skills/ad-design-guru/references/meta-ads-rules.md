# Meta Ads — Platform Rules & Best Practices

## Format Specifications

### Feed Ads (Facebook + Instagram)

**Square 1:1 — 1440×1440px**
- Most versatile format
- Works in Facebook Feed, Instagram Feed, Marketplace, Messenger
- Safe for carousel ads (same dimensions per card)
- Padding: minimum 50px from all edges

**Portrait 4:5 — 1440×1800px**
- Recommended for mobile-first campaigns
- Takes up more screen real estate in feed = higher stopping power
- Instagram Feed preferred format
- Extra vertical space is ideal for product shots or lifestyle imagery

**Story/Reel 9:16 — 1080×1920px**
- Full-screen immersive format
- TOP SAFE ZONE: Avoid top 250px (username, timestamp overlay)
- BOTTOM SAFE ZONE: Avoid bottom 250px (CTA button, swipe-up, message)
- CENTER SAFE AREA: 1080×1420px is your actual working canvas
- Design for sound-off by default (captions, visual storytelling)
- First 3 seconds must hook — front-load the message

**Landscape 1.91:1 — 2064×1080px**
- Used for link ads, right column, Audience Network
- Less mobile screen real estate — must be punchy and simple
- Headline and CTA must work at small sizes
- Best for direct response / traffic campaigns

### Carousel Ads
- 2-10 cards, each 1:1 (1440×1440px)
- Each card should work standalone AND as part of a sequence
- First card is the hook — most important
- Last card should be the CTA card
- Maintain visual consistency across cards (same style, colors, typography)

## Text Overlay Guidelines

Meta's 20% text rule is no longer a hard rejection, but:
- Ads with LESS text get better delivery
- Ads with MORE text get higher CPM (cost per thousand impressions)
- Design principle: let the image do the work, put copy in ad text fields

**What counts as text:**
- Headlines and body copy on the image
- Text in logos (except very small logos)
- Watermarks with text
- Numbers and prices

**What doesn't count:**
- Text in product photos (e.g., label on a wine bottle)
- Infographics where text IS the content

## Creative Best Practices

### Attention & Stopping Power
- Use high-contrast visuals against the feed's white/dark background
- Faces and people increase engagement
- Movement/dynamic angles outperform static product shots
- Bright, saturated colors stop the scroll

### Mobile-First Design
- 98% of Facebook users access via mobile
- Design for thumb-scrolling speed (you have 1.7 seconds in feed)
- Large text that's readable without zooming
- Simple, uncluttered layouts

### CTA Best Practices
- Use action verbs: "Shop", "Discover", "Get", "Try", "Start"
- Create urgency when appropriate: "Limited time", "Today only"
- Visual CTA button on the image reinforces the platform CTA button
- Place CTA in bottom-right (Z-pattern ending point) for landscape/square
- Place CTA in lower-third for portrait/story

### What NOT to Do
- Don't use stock photo watermarks
- Don't cram multiple messages into one ad
- Don't use tiny text that's unreadable on mobile
- Don't place key content where Meta UI elements overlay
- Don't use misleading imagery or clickbait visuals
- Don't forget to check how the ad looks in dark mode
